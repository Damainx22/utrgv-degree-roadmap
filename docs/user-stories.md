# User Stories

## Authentication
- As a student, I want to register an account so that my progress is saved.
- As a student, I want to log in securely so that only I can access my data.

## Degree Planning
- As a student, I want to select my major so that the app knows my degree requirements.
- As a student, I want to input completed courses so that I can track my degree progress.
- As a student, I want to see remaining required courses so that I know what I still need to take.
- As a student, I want to see which courses I am eligible to take next based on my prerequisites so that I don't register for a class I'm not qualified for.
- As a student, I want to see a semester-by-semester roadmap so that I can plan my path to graduation.

## Schedule Builder
- As a student, I want to select courses I want to take next semester so that I can build a schedule.
- As a student, I want to filter classes by time of day so that I can build a schedule that fits my life.
- As a student, I want the app to generate a conflict-free weekly schedule so that I don't have overlapping classes.

## Professor Reviews
- As a student, I want to see professor ratings alongside each course section so that I can make informed registration decisions.
- As a student, I want to read reviews from other students so that I know what to expect from a professor.
- As a student, I want to submit a review for a professor so that I can help other students.
