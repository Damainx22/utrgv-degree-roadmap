# User Stories

## Account System
As a student, I want to create an account so that I can save my academic progress.

As a student, I want to log into my account so that I can access my roadmap.

As a student, I want to update my profile information.

## Degree Planning
As a student, I want to input completed courses so I can track my degree progress.

As a student, I want to see remaining required courses so I know what to take next.

As a student, I want the system to generate a suggested roadmap so I can graduate on time.

## Professor Reviews
As a student, I want to read professor reviews so I can choose instructors.

As a student, I want to submit professor reviews to help other students.
